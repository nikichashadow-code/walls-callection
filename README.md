## Installation & Usage

### 📥 Getting Started

1. **Clone the Repository**
   ```powershell
   git clone https://github.com/nikichashadow-code/walls-callection.git
   cd walls-callection
   ```

2. **Browse the Collection**
   - Navigate through the category folders to find wallpapers you like
   - Each folder contains wallpapers organized by theme

3. **Set as Wallpaper (Windows)**
   - Right-click on any image file
   - Select "Set as desktop background" or "Set as lockscreen"
   - Alternatively, use Settings > Personalization > Background

4. **Set as Wallpaper (macOS)**
   - Open System Preferences > Desktop & Screen Saver
   - Click the "+" button and select your wallpaper
   - Choose how to fill your screen

5. **Set as Wallpaper (Linux)**
   - Method varies by desktop environment
   - GNOME: Right-click desktop > Change Background
   - KDE: Right-click desktop > Configure Desktop

### 🔄 Regular Updates

This collection is maintained and updated regularly. To keep your local copy current:

```powershell
git pull origin main
```

### 💾 Downloading Specific Categories

To download just one category:
```powershell
git clone --depth 1 --filter=blob:none --sparse https://github.com/nikichashadow-code/walls-callection.git
cd walls-callection
git sparse-checkout set anime-walls
```

## File Formats

### PNG Format
- **Extension:** `.png`
- **Best for:** Images requiring transparency or lossless quality
- **File Size:** Larger than JPG
- **Quality:** Lossless compression
- **Use Cases:** Wallpapers with transparent elements, artistic designs

### JPG Format
- **Extension:** `.jpg`, `.jpeg`
- **Best for:** Photographs and images with many colors
- **File Size:** Smaller than PNG
- **Quality:** Lossy compression (minimal visible quality loss)
- **Use Cases:** Photographs, scenic images, landscapes

### Recommendations
- **For Maximum Quality:** Use PNG files
- **For Smaller File Sizes:** Use JPG files
- **For Memory/Storage:** JPG format is more efficient
- **For Artistic Work:** PNG format preserves quality and transparency
